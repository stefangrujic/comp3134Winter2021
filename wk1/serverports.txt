1) What is the definition of server port?

A number to identify what process the server should use's when a request is sent.

2) What is server port range?

0 - 1023, Well Known ports
1024 - 49151, Registerd ports
49152 - 65535, Dynamic ports

3) How many bits does number above represent?
16-bit 

4) State 10 common server ports

80 HTTP, 443 HTTPS, 20 FTP Data, 21 FTP Control, 22 SSH,
23 Telnet, 25 SMTP, 53 DNS, 110 POP3, 161 SNMP
